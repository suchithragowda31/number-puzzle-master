# number-puzzle-master
# React Native Game

A number-matching puzzle game built with React Native and Expo, inspired by the Number Master game.

## Game Overview

Match identical numbers or pairs that sum to 10 by connecting them through clear paths. Complete each level by achieving the target number of matches within the time limit.

##  Setup & Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Expo CLI

### Installation Steps
```bash
# Clone the repository
git clone https://github.com/suchithragowda31/number-puzzle-master.git
cd number-puzzle-master

# Install dependencies
npm install

# Install required packages
npx expo install react-native-safe-area-context

# Start the development server
npx expo start
